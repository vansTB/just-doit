# just do it

## introduce

- 一款番茄时钟webApp，拥有以下功能
  - 用户列表
    - 用户名
    - 密码
    - 创建时间
    - 邮箱提醒状态
    - 邮箱提醒时间
  - 登录
- 学习主题（默认主题 * 1）
  - 主题名称
  - 计时方式
    - 正计时
    - 倒计时
  - 创建时间
  - 备注
- 学习记录
  - 创建时间
  - 完成状态
  - 持续时间
  - 备注

## Client

- react18

- react-router
- tailwindCss
